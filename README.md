# 💰 Budget Tracker App

A sleek, functional, and responsive **Budget Tracker Application** designed to help users manage their finances effectively. Users can set a total budget, add specific expenses with titles and costs, and dynamically track their total budget, total expenditures, and remaining balance in real-time.


---

## ✨ Features

* **Dynamic Budget Management:** Set a total budget and update your financial limits instantly.
* **Expense Tracking & Logs:** Add daily expenses with a product/service title and its cost.
* **Real-time Financial Summary:** Live calculations of Total Budget, Expenses, and remaining Balance.
* **CRUD Operations (Edit & Delete):** Modify or delete existing expenses from the list, with automated real-time recalculations of the total balance and expenditure fields.
* **Input Validation & Error Handling:** Prevents empty submissions, negative values, or invalid inputs via clear UI error feedback.
* **Responsive Layout:** Fully optimized for all screen sizes (Mobile, Tablet, and Desktop) using modern CSS Grid and Flexbox layout methods.

---

## 🛠️ Tech Stack & Architecture

This project was built with an emphasis on **clean code** and **strong typing** principles:

![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=flat-square&logo=html5&logoColor=white)
![CSS3](https://img.shields.io/badge/CSS3-1572B6?style=flat-square&logo=css3&logoColor=white)
![TypeScript](https://img.shields.io/badge/TypeScript-3178C6?style=flat-square&logo=typescript&logoColor=white)

* **Semantic HTML5:** Structured for high accessibility and SEO-friendly document flow.
* **Modern CSS3:** Features a responsive layout using custom font embedding (Poppins), explicit box-shadow layers for depth, and robust grid configurations.
* **Strictly Typed TypeScript:** Built with reliable strict DOM type casting (`HTMLInputElement`, `HTMLButtonElement`), explicit function return types (`void`), and structured array manipulations for maximum runtime predictability.
